# Project Vision

## Working Identity
`Parry Shooter` hiện là một prototype shmup dọc 1 tay, lấy parry làm core differentiator thay vì chỉ né và bắn.

## Core Fantasy
Người chơi không chỉ sống sót giữa đạn, mà còn:
- chủ động lao vào đúng viên parry
- biến nguy hiểm thành phản công
- dùng build/loadout để đổi cách phản công đó

## Current Prototype Priorities
Prototype hiện đang ưu tiên:
- một tay, kéo để di chuyển
- auto shot
- parry gesture upward
- boss-rush-first iteration
- tutorial riêng để dạy mechanic
- data-driven loadout cho player
- data-driven move authoring cho boss

## Current Playable Layers
### 1. Core Control
- movement feel
- pointer offset
- parry burst
- grace timing

### 2. Combat Systems
- HP
- Rage
- skill
- boss break / weak zone
- counter projectile
- parry special

### 3. Authoring
- player loadout SO
- player character roster SO
- boss move definition SO
- localization asset
- audio catalog

### 4. UX / Flow
- boss rush start panel
- character selection
- tutorial scene riêng
- settings panel
- language + volume persistence

## Current Design Principle
Feel quan trọng hơn số lượng feature.

Những gì đã được chốt feel trong prototype:
- movement response
- one-hand control philosophy
- parry activation feel
- counter shot bay thẳng trục Y

## Near-Term Direction
Prototype hiện không cố mở rộng thành full game ngay. Mục tiêu gần là:
- ổn định combat loop
- dễ authoring thêm move / loadout / character
- test nhanh boss pattern
- giữ tài liệu bám sát code thực tế
